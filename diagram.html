Flujo de datos del proyecto Course Sync:

  1. Fuentes de datos: Se obtienen cursos desde las APIs de Udemy y Pluralsight
  2. Procesamiento: Los datos se normalizan a un formato unificado y se comparan con los datos existentes en Eightfold
  3. Exportación: Se generan archivos CSV y XML con los datos procesados
  4. Destino: Los archivos se suben mediante SFTP a Eightfold para actualizar el catálogo de cursos
